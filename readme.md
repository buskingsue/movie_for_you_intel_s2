# 영화 추천 앱
##
## 이 코드는 PyQt5를 사용하여 만든 영화 추천 애플리케이션입니다.
## 영화 제목이나 키워드를 입력하면 해당하는 영화와 유사한 영화를 추천해주는 시스템입니다. 

## tf와 df의 차이를 알려줘
## TF (단어 빈도)	DF (문서 빈도)
##
## pip freeze > requirements.txt
